Dotfiles
========

Files that make my life easier and more colorful

##Vim 


To use vundle:

Install vundle:

```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```


and then launch vim and run:

```
:PluginInstall
```
